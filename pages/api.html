API

Supported APIs

  • Current Hashrate – current estimated network hashrate
  • Current Height – current height (incl. genesis block)
  • Block Reward – current block reward
  • Circulating Supply – circulating supply
  • Total Supply – current total supply of coins
  • Total Banked – current total banked coins
  • Max Supply – Max supply for the coin
  • Current Difficulty – current difficulty
  • All Node Count – current nodes count
  • Smart Node Count – current smart nodes count
  • ETH wCCX Circulating Supply – current smart nodes count
  • ETH wCCX Total Supply – current smart nodes count
  • BSC wCCX Circulating Supply – current smart nodes count
  • BSC wCCX Total Supply – current smart nodes count
  • wCCX Total Supply – current smart nodes count
